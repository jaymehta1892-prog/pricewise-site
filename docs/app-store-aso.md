# Pricewise — App Store Optimization (ASO) pack

Paste these into **App Store Connect**. App Store keywords live in the store, not on your website, so this is separate from the site's SEO.

**Rules followed (2026):** App Name ≤30 chars · Subtitle ≤30 · Keyword field = 100 chars, comma-separated, **no spaces** · singular words only (Apple auto-matches plurals) · never repeat a word across Name/Subtitle/Keywords · no brand or competitor/retailer names in the keyword field.

## App Name — 30 chars
```
Pricewise: Grocery Price Match
```

## Subtitle — ≤30 chars (pick one)
```
Save on flyer deals & lists        (27)
Flyer deals & savings near you      (30)
```

## Keyword field — exactly 100 chars (no spaces)
```
coupon,discount,cheap,weekly,sale,supermarket,shop,compare,receipt,rebate,toronto,ontario,gta,budget
```

## Promotional text — ≤170 chars (editable anytime, no search impact)
```
Turn your grocery list into savings. Pricewise finds price-match deals on items you already buy and points you to the nearest GTA store that'll match them — free.
```

## Notes
- App Name / Subtitle / Keyword field changes require your **next build submission** (your keyword field is locked to the current live build). **Promotional text can be updated right now.**
- Do **not** add retailer names (No Frills, Walmart, Metro, etc.) or competitor app names to the keyword field — Apple can reject for trademark use.
- Biggest expansion lever: add **localized keyword sets** (e.g., a French-Canada locale) — each locale gets its own 100-char field, multiplying your indexed terms.
